# 相机标定

## 注意事项

- 输入 路径-- default.xml

- 输出路径 -- out_camera_data.xml

- src/app/camera是调用相机保存图片到images下

- src/app/cali是相机标定(OpenCV例程)

## 热键

|热键|作用|
|------|------|
|q|退出|
|v|切换录像开始关闭|
|空格|截图|
